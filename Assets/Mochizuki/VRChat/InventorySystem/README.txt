# VRChat Inventory System for Unity 2018

Unity 2018.3 以降、 VRChat 2020.1 以降 (Unity 2018 対応版以降) で使用できる、C# スクリプトを使わないインベントリシステムです。
比較的簡単にセットアップできる Unity エディター拡張付きです。

インベントリ1つ実装するに当たり、以下の要素を使用します。

* Particle System 3個
* Particle Count 3個
* Animator 7個
* Collider 最低1個

そのため、小数の設置でも Avatar Performance が Poor になる可能性があります。

なお、プロジェクト本体は GitHub にて公開されており、知識があれば誰でも無料で使用することが可能です。
https://github.com/mika-f/VRChat-InventorySystem


## 試用版について

本製品は無料の試用版を提供しています。
製品版 (有料) を購入する際は、あらかじめ試用版を導入し、動作に問題がなかった事を確認したあと購入するのをお勧めします。
なお、試用版は製品版と比べて、以下の要素が除外されています。

* DefaultOFF (アバター初期化時点で非表示) の Prefab および関連アセット

なお、試用版で十分だった場合は、製品版を購入する必要はありません。


## 制限事項

以下の状況において、 VRChat の仕様上自身に見えているメッセージと相手に見えるメッセージが異なる可能性があります。
あらかじめご了承ください。

* 表示・非表示を切り替えた時点で、見せた相手がワールドにいなかった場合
* 表示・非表示を切り替えた時点で、見せた相手が自身を見ていなかった、もしくは一定距離離れていた場合

(EmoteSwitch とほぼ同様だと思って頂いてかまいません。)


## 注意事項

不具合などがあるかもしれません。ご了承ください。

なお、以下の環境にて動作を確認しています。

* Unity 2018.4.20f1
* VRChat 2020.2.2 Build 932
* VRCSDK2 2020.04.25.12.49


## 他のインベントリシステムとの違い

できること

・任意の場所に無制限に設置が可能です。
・初期状態を選択することができます。
・Unity 2018.4.20f1 で制作されています。
・エディター拡張が付属しているので、簡単にセットアップができます。

1つめについては、例えば、キッシュちゃんのカバンにくっつけることができます。
https://twitter.com/6jz/status/1256876847679172608

2つめについては、この動画のように帽子の切り替え機能を実装することが可能です。
https://twitter.com/6jz/status/1257291906561609733


できないこと

・他のアイテムを取り出したとき、自動で今だしているものをしまう機能はありません (トグル機能)。


## GitHub 版との違いについて

BOOTH ではビルド済みの UnityPackage と説明書を付けたパッケージを配布しています。
GitHub 版でもほぼ同様のものは作成可能ですが、サポートなどは行いません (自己解決出来る人向けです)。
